New SEO Tools Project

Nov 14th, 2012

As you may or may not know I’m an SEO and often write about SEO and Marketing on my website. I decided to start working on a set of tools to put on my site, and I’ve open sourced them so others can use them and hopefully fork or improve upon.

The repo is located here and I have a rough start going so far, you can also access the live demo on IronFoundry. I hope to add some more cool stuff to this project in the future.

This is a collection of SEO tools for use on an ASP.Net website running at least the 3.0 .Net framework. I only have two tools for now, but plan to add more.

We currently have

Blog Pinger - Pings multiple blog update sites when you've updated your blog
Server Header Checker - Check your server headers for diagnostic purposes

Future plans:

Code to text ratio tool
html compressor
image optimizer
keyword analyzer
backlink checker

This project is open to any suggestions or contributions. If you want to change or add anything send a pull request! I hope to be adding more stuff soon

Octopress Header Adder

Oct 27th, 2012

I just created the Octopress Header Adder today.

This is just a simple ruby script that adds some generic headers for Octopress. Let’s say you have a bunch of files from Wordpress or another blog that you created from markup, you can use this utility to add header information to generate Octopress pages.

It’s intended for those who convert all their html files to markup, it’s a convienent way to drop some headers into it.

To run it, simply type

In the folder with your markdown files. It will add a generic header to the beginning of all of them.

I wrote this for someone who had emailed the Octopress mailing list about it, and figured I would share it here, if one person benefits from it then it’s worth the 10 minutes I put into it.

Feel free to branch or shoot me ideas for improving it.

PHP Ajax Dropdown Demo

Sep 22nd, 2012

I recently created a tutorial on my site for building a PHP Ajax dropdown list and detailed some of the steps required so people could get a good understanding of how AJAX works. Naturally these days most people would simply use jQuery or one of the tons of other quality frameworks for something so simple, but the purpose of this was to teach the basics and show you how AJAX works.

You can get the source here and if you fork it or make a more detailed tutorial let me know! I’d love to see what you’ve done. Please don’t use this on a production webserver as it is for demonstration and education only, it’s highly insecure and very basic.
