New SEO Tools Project

Nov 14th, 2012

As you may or may not know I’m an SEO and often write about SEO and Marketing on my website. I decided to start working on a set of tools to put on my site, and I’ve open sourced them so others can use them and hopefully fork or improve upon.

The repo is located here and I have a rough start going so far, you can also access the live demo on IronFoundry. I hope to add some more cool stuff to this project in the future.

This is a collection of SEO tools for use on an ASP.Net website running at least the 3.0 .Net framework. I only have two tools for now, but plan to add more.

We currently have

Blog Pinger - Pings multiple blog update sites when you've updated your blog
Server Header Checker - Check your server headers for diagnostic purposes

Future plans:

Code to text ratio tool
html compressor
image optimizer
keyword analyzer
backlink checker

This project is open to any suggestions or contributions. If you want to change or add anything send a pull request! I hope to be adding more stuff soon
