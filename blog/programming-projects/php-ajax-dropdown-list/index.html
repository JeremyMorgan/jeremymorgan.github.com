PHP Ajax Dropdown Demo

Sep 22nd, 2012

I recently created a tutorial on my site for building a PHP Ajax dropdown list and detailed some of the steps required so people could get a good understanding of how AJAX works. Naturally these days most people would simply use jQuery or one of the tons of other quality frameworks for something so simple, but the purpose of this was to teach the basics and show you how AJAX works.

You can get the source here and if you fork it or make a more detailed tutorial let me know! I’d love to see what you’ve done. Please don’t use this on a production webserver as it is for demonstration and education only, it’s highly insecure and very basic.
